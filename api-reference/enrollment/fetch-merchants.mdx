---
openapi: get /api/merchants
---

Returns all merchants for your Chargeblast account. The response is **paginated** using the `page` and `per` query parameters.

Each merchant in the response includes a **name** and **id**. The `name` is set during onboarding and, for multi-merchant accounts, corresponds to the individual merchant name (typically the DBA or billing descriptor).

**Example response:**

```json
{
    "page": 1,
    "total": 1,
    "merchants": [
        {
            "name": "sandbox-merchant-1",
            "id": "cb_xxxxxxxxxxxxx"
        }
    ]
}
```
