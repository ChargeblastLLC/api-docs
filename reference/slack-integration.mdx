---
title: "Slack Integration"
description: "In order to set up the Slack Integration:"
mode: wide
icon: slack
iconType: solid
---

1. In the [integrations](https://app.chargeblast.io/integrations) page, select the Add to Slack button
<Frame>
  <img src="/images/reference/slack-add-to-slack-integration.png"/>
</Frame>

1. Type in the channel you want to send alerts to (without the #).
2. If the channel is a private channel, make sure to add the Chargeblast bot on Slack to the private channel. You can do this by just tagging @chargeblast in the private channel.
<Frame>
  <img src="/images/reference/slack-channel-selection-chargeblast.png"/>
</Frame>

1. Press the "test notification" button to ensure the integration is working.
